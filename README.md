# Векторный редактор CLI
Простой векторный редактор с интерфейсом командной строки (CLI) для работы с фигурами

## Описание

В приложении реализовано:
- Создание геометрических фигур (точка, линия, круг, квадрат)
- Удаление созданных фигур по прядковому номеру
- Просмотр списка всех созданных фигур
- Интерактивно работать через командную строку

## Стек

- Python 3.12
- Pytest

## Установка

1. Клонирвание репозиторий
```bash
git clone https://github.com/Mr-Dron/test_neiros.git
```

2. Устанавливаем зависимости
```bash
pip install --no-cache-dir -r requirements.txt
```

## Использование
запуск приложения
```bash
python3 main.py
```

## Структура проекта
```bash
test_neiros/

    models.py         # Классы геометрических фигур
```
